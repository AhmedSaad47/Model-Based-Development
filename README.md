## 👤 About This Repository

This repository documents my personal journey through the **Model-Based Development (MBD) Diploma** by OS Academy. It serves as a living record of my progress, notes, exercises, and project work completed throughout the program — and as part of my professional portfolio.

The diploma covers the full MBD workflow: from MATLAB & Simulink fundamentals through to embedded code generation and a real-world Battery Management System capstone project. The skills developed here are directly applicable to roles in **automotive, aerospace, robotics, and industrial control systems**.

---

## 🛠️ Skills & Tools Covered

| # | Topic | Description |
|---|---|---|
| 1 | 🧠 **MBD Concepts** | Foundational principles of model-based development and how models drive system design |
| 2 | 📊 **MATLAB Basics** | Data analysis, visualization, and algorithm development |
| 3 | 🔧 **Simulink Basics** | Dynamic system simulation and control system design |
| 4 | 🤖 **Arduino Support Package** | Interfacing MATLAB/Simulink with Arduino hardware for embedded systems |
| 5 | 🎛️ **PID Controller Design** | Designing and tuning Proportional-Integral-Derivative controllers |
| 6 | 📝 **MATLAB Scripting** | Automating tasks, writing custom functions, and improving workflow efficiency |
| 7 | 🏗️ **System Modeling** | Representing system requirements, architectures, and behaviors using models |
| 8 | 🧩 **Model Architecture** | Structuring complex models for scalability and clarity |
| 9 | 🔀 **Stateflow Design** | Modeling discrete event systems and finite state machines |
| 10 | ⚙️ **Embedded Coder & Code Generation** | Generating production-quality C/C++ code directly from Simulink models |
| 11 | 💾 **C Programming Overview** | C programming fundamentals for embedded applications and deployment readiness |
| 12 | 🔋 **Battery Management System (BMS) Project** | Capstone project: modeling and simulation of a full BMS system in Simulink |
